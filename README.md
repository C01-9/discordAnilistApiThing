# discordAnilistApiThingy
a discord bot written in python that uses the anilist api. I made this mostly cuz i was bored and it has no practical use.

to got it to work you'll need to make a bot in the [discord developer portal](https://discord.com/developers/applications). You'll also need the [discord.py](https://discordpy.readthedocs.io/en/latest/index.html) and the [requests](https://pypi.org/project/requests/) libraries.
